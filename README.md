# Technologies Installer

These scripts install the stuff one might require for their development Endavours.

## Install NodeJS

`cd path\to\installer`

Then: `sh node.sh`

## Install Cassandra

`cd path\to\installer`

Then: `sh cassandra.sh`

## Install Redis

`cd path\to\installer`

Then: `sh redis.sh`

## Install LAMP on Ubuntu 14.04

`cd path\to\installer`

Then: `sh lamp.sh`
